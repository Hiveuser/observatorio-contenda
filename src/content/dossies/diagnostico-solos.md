# Dossiê: Caracterização Pedológica de Contenda/PR

## Classes de Solo (por % de área)

| Classe | Sigla | % Área | Aptidão Principal |
|--------|-------|--------|-------------------|
| Latossolo Vermelho-Amarelo | LVA | 45% | Lavoura / pastagem |
| Podzólico Vermelho-Amarelo | PV | 30% | Lavoura / pastagem |
| Cambissolo | C | 15% | Pastagem / florestal |
| Gleissolo / Hidromórfico | G | 10% | Preservação / APP |

## Características

- **Textura:** Argilosa a muito argilosa (40–70% argila)
- **pH médio:** 4,8–5,5 (ácido) — necessita calagem
- **Matéria orgânica:** 2–4% (adequado para Cfb)
- **Erosão:** Moderada em áreas sem cobertura — relevo ondulado
- **Problema crítico:** Compactação em áreas de pastagem intensiva

## Recomendação de SAF por Classe

- **LVA (45%):** Faixas de milho × bracatinga × guanandi
- **PV (30%):** Calagem + correção de Al³⁺, depois SAF diversificado
- **Cambissolo (15%):** Pastagem rotacionada com faixas de floresta
- **Gleissolo (10%:** PRESERVAÇÃO — drenagem inadequada para uso agrícola
