# Cobertura Florestal de Contenda/PR — MapBiomas 2023

## Dados

- **Cobertura florestal remanescente:** 38% do território municipal
- **Bioma:** 100% Mata Atlântica
- **Fragmentação:** Alta — fragmentos isolados insuficientes para 70% das propriedades
- **Principal ameaça:** Supressão irregular e expansão pecuária sem compensação

## Contexto Legal

Contenda está integralmente no domínio do bioma Mata Atlântica (Lei 11.428/2006).
A RL obrigatória para pequenas propriedades é de 20% (Lei 12.651/2012), mas
30% das propriedades cumprem o requisito.

## Recomendação

- Recomposição via SAF de bracatinga + guanandi + milho — economicamente viável
- Priorizar corredores ecológicos entre fragmentos
- Integrar com programa CAR/RL para acelerar regularização
